# GeoLookup
IP to GeoLocation Lookup using MaxMind GeoIP2 API

## Require
MaxMind.GeoIP2 nugetから入手してください

`Install-Package MaxMind.GeoIP2` use nuget

## Usage 
`GetLookup [IP ADDRESS | Lookup List file]`

## 注意
エラーハンドリング等まだ未実装です
